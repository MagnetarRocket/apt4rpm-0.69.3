Element excludedir

Name

excludedir

Synopsis

Mixed Content Model 
excludedir ::=
( #PCDATA )
Attributes
None

Description

Define directories to excluded from download

This is a wget option (-X) and specifies which directories of the download server should not be retrieved. The directories may contain wildcards. The directory should be the full path, starting just right after the hostname up to and including the directory to be excluded.

Parents

These elements contain excludedir: wget .
