Aptate's top (or root) directory determines, where the apt repository(ies) will reside. All other directories are relative to this directory. A directory called apt will be created by aptate under this directory. The apt directory holds the apt repositories and directories that are used by aptate internally (see the description of the element <sharedir>).

The following characters are allowed: alphanumeric -_./

Examples are: /dist, /distribution/okna2001, etc.
