Description

Define the shared directory.

The sharedir tag defines the location of the shared directory. In this directory several other directories are stored that are used by aptate internally. The sharedir defaults to <topdir>/apt/share and in most, if not in all cases this is the right value to use. It is possible to specify an absolute path.

Information that is stored in this directory are caches, the rpms that are mirrored from download servers and the download log files. The directories are organized by distribution id.

The following characters are allowed: alphanumeric -_./
