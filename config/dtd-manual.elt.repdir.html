Synopsis

Mixed Content Model 
repdir ::=
( #PCDATA )
Attributes
None

Description

Define the repository directory.

The repdir tag defines the location of the apt subdirectory below $topdir. $topdir/$repdir is the directory aptate will put its files into. Default: apt.

Setting repdir to an empty string will cause aptate to put its files into $topdir directly.
