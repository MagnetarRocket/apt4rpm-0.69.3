add-arg ::=
( #PCDATA )
Attributes
None

Description

Define additional wget arguments

This tag can be used to specify any wget option. These options are transparently handed over to wget. For more information about the options available, see the wget manual.

Example:
-reject *-devel-* -proxy=off
