Description

Provide an example sources.list ftp url

The information provided by this element is used to create an example /etc/apt/sources.list file for the ftp protocol. The information is the url that describes how your apt repository users can reach your repository via the ftp protocol. An example of such a url is: ftp.okna98.rm/apt (it's no problem if the protocol is included in the url).
